App to create a shopping list of all the items you want to buy either it's online or at a grocery store, department store, etc.
The program will do a tally of the item(s) with it's pricetag and the tax %.
At the end it will give you a total count of all items with the total price (PreTax and with tax)
User could "save" the list items to an outfile "listbox_items.txt" for re-use later.
User could edit each item.
User could filter items by typing a few starting characters.
